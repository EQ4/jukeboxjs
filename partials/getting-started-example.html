Getting Started Example

This example will create a synthesizer and play it.

This example uses two optional but recommended dependencies, the schema pack and the shim. 

<script src="scripts/schema.js"></script> <!-- Include the schema packs for fast configuration -->
<script src="scripts/web-audio-shim.js"></script> <!-- Improve compatibility -->
<script src="scripts/jukebox.js"></script>

<script>
  var schema = JBSchema.synthesizers["Omaha DS6"]; // The schema pack adds a global object.
  var synth = Jukebox.getSynth(schema); 

  synth.play(10,1000) // bing;
</script>
