API

Linking to the script creates a global Jukebox class

Jukebox

Jukebox():Jukebox Returns a new Jukebox instance. Each jukebox has its own audio context. Create a new jukebox instance to begin.
jukebox.Synth():Synthesizer Returns a new synthesizer object.
jukebox.Drums():Drums Returns a new drum object.

Synthesizer

synthesizer.tone(frequency:Number,duration:Number):null

Plays a tone of the specified frequency for the specified duration of time.

A synthesizer can play only one tone at a time. To play multiple tones at once, create more than one synthesizer.

synthesizer.sequence([{ frequency:Number , duration:Number }...])

Plays a sequence of tones one after the other.

Example

Create a synthesizer 
var jukebox = new Jukebox();
var moog = new jukebox.Synth();
Play a tone 
moog.tone(440,300);
Play a harmony 
var moog = new jukebox.Synth();
var korg = new jukebox.Synth();
moog.tone(392,300);
korg.tone(262,300);
Play a sequence 
moog.sequence([{
  frequency:294,
  duration:500,
},{
  frequency:330,
  duration:500,
},{
  frequency:262,
  duration:500,
},{
  frequency:131,
  duration:500,
},{
  frequency:196,
  duration:500,
}]);

Drums

Creates an instrument that creates percussive noises.

drum.tone(tone:int):null

Plays a tone corresponding to the specified integer as configured by the individual drum set.

Recommended values are as follows - 0: kick drum, 1: snare, 2: hi-hat, 3: cymbal
